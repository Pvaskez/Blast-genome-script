# Blast-genome-script
Automates BLAST searches to identify and extract gene sequences from multiple genomes. Builds BLAST databases, filters hits (>70% identity, >1600 bp), and retrieves matching contigs into FASTA files for cross-species gene comparison and copy-number analysis.
